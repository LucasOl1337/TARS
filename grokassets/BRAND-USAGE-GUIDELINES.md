## Recommended File Usage
- Primary logos: Use horizontal dark version for most digital contexts. Light versions available for bright backgrounds.
- App icons: Use the 1024px PNG as master. Generate other sizes from it.
- Social banners: The 1500x500 versions are optimized for X/Twitter. Crop or regenerate for other platforms as needed.
- OG images (1200x630): Use for link previews on all major platforms.
- Pitch deck backgrounds: Designed with generous safe margins for text overlays.
- Content illustrations: High-resolution masters suitable for web and print.

## Do's and Don'ts
- Always respect the project's specific visual DNA (detailed in visual-bible.md).
- Do not mix assets across different projects without clear separation.
- Maintain consistent margins and breathing room as shown in generated masters.
- For video tools: Use the provided thumbnail/lower-third/end-screen templates as starting points and customize per video.
